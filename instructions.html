Sudoku Solver

How To Play

About :

This is an app which lets us play the game of Sudoku. The unique feature of this app is that it lets us visualize the Bactracking process that is used to solve the board.

How to play :

  • The first step is to enter the numbers into the board. We can use the number pads on screen, or we can use our keyboards to enter the numbers. We can also use the arrow keys to move around easily.
  • After entering the initial numbers, we can click Done. If the board is unsolvable / invalid, we'll need to fix it before moving on. If the board is valid, we can start solving.
  • We can enter the numbers using the keyboard or the number pad on screen.
  • If the entered number is correct, it will be locked and we can't change it again.
  • If the entered number is wrong, it will blink red and go blank.
  • Once the board is solved, we can click on New Board to clear everything and start a new game.

Visualizing Backtracking :

  • At any point, if we want to visualize the backtracking process, we can click the Solve button and the app starts solving the board using the backtracking algorithm.
  • We can control the speed of the animations by simply changing the slider on top.
  • Once the animations are done, we will have a solved board!
  • Once the board is solved, we can click on New Board to clear everything and start a new game.
